Everything you need to set Claude up for your business in one place: templates, prompts, workflows, and strategies you can use today. No fluff, no theory. Each file is a working asset. Grab the one you need and start.

## Get it

```bash
git clone https://github.com/treycooper3/claude-starter-kit.git
```

Or click the green **Code** button above and **Download ZIP**. Everything is plain Markdown, so it opens anywhere.

Start with the printable overview: [Claude-Starter-Kit-Overview.pdf](Claude-Starter-Kit-Overview.pdf).

| File | What it gives you |

| [02 - Claude Code Quickstart](kit/02-claude-code-quickstart.md) | Set up Claude Code (the terminal AI) with a CLAUDE.md file |

| [04 - Prompt Library](kit/04-prompt-library.md) | 30 ready-made prompts for common business tasks |
| [05 - Workflow Templates](kit/05-workflow-templates.md) | 6 SOP templates built on the WAT framework |

## Where to start

- **New to Claude?** Open `01 - Claude.ai Quickstart`. Personalized setup in 15 minutes.
- **Want prompts right now?** Go straight to `04 - Prompt Library`. Copy, customize, run.
- **Technical builder?** Start with `02 - Claude Code Quickstart`. That is the power tool.
