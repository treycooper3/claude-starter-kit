# The Claude Starter Kit

> Claude is a tool. This kit makes it YOUR tool. Everything you need to set Claude up for your business in one place: templates, prompts, workflows, and strategies you can use today.

---

## What it is

A free, copy-paste starter pack that takes you from "I have a Claude account" to "Claude runs real work for my business." No fluff, no theory. Six short files, each one a working asset. Grab the one you need and start.

## What's inside

| File | What it gives you |
|------|-------------------|
| **01 - Claude.ai Quickstart** | Set up Claude.ai (web) with custom instructions in 15 minutes |
| **02 - Claude Code Quickstart** | Set up Claude Code (the terminal AI) with a CLAUDE.md file |
| **03 - Custom Instruction Templates** | 8 copy-paste templates by business type. Grab the one that fits |
| **04 - Prompt Library** | 30 ready-made prompts for common business tasks. Instant value |
| **05 - Workflow Templates** | 6 SOP templates built on the WAT framework (Workflows, Agents, Tools) |
| **06 - Strategies & Patterns** | Advanced moves to get production-grade output, not generic answers |
| **Bonus - Single-Prompt Video** | One master prompt that turns a topic into a full faceless video package |

## What it does for you

- **Cuts the setup learning curve to one sitting.** You stop guessing and start with a configured Claude.
- **Turns prompts into assets.** The library and templates are reusable, not one-off chats.
- **Gives you a repeatable system.** The WAT workflow SOPs make Claude reliable, not just impressive.

## Where to start

- **New to Claude?** Open `01 - Claude.ai Quickstart`. You will have a personalized setup in 15 minutes.
- **Want prompts right now?** Go straight to `04 - Prompt Library`. Copy, customize, run.
- **Technical builder?** Start with `02 - Claude Code Quickstart`. That is the power tool.

## Claude.ai vs Claude Code (the 10-second version)

- **Claude.ai (web):** chat for writing, research, and analysis. No code needed.
- **Claude Code (terminal):** reads and edits your files, runs scripts, builds software. The game-changer once you are building.

Many members use both. The kit sets up each one.
